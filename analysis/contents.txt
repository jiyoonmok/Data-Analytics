1. 데이터 분석 라이브러리
    (1) numpy
        - 벡터 및 행렬 연산과 관련된 편리한 기능을 제공
        - 특히 배열과 관련된 연산을 제공
        - 여러가지 다양한 수학함수 제공
    (2) pandas
        - Series, DataFrame 등의 자료구조를 활용하여 데이터 분석에 있어서 우수한 성능을 발휘
        - 대량의 데이터를 더욱 빠른 속도로 처리
        - 데이터 분석의 메인 도구
    (3) matplotlib
        - 시각화
        - Seaborn 
        
     *. scipy
         - 과학, 분석용 오픈소스 기반의 라이브러리
         - numpy array object 위에서 구축, 개발됨
